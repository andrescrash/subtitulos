⚠️ Chrome Translation APIを有効にする方法

  1. Chrome 138以降を使用していることを確認
  2. アドレスバーに chrome://flags を入力
  3. #translation-api を検索して「Enabled」に設定
  4. #language-detection-api を検索して「Enabled」に設定（オプション）
  5. Chromeを再起動

注意: これは実験的な機能です。将来のバージョンで変更される可能性があります。
